# Assets

This folder contains media assets for the README.

## demo.gif

To create a demo GIF:

1. Run the driver: `python src/driver.py`
2. Record your screen while pressing buttons
3. Convert to GIF and save as `demo.gif`

Recommended tools:
- [Kap](https://getkap.co/) - macOS screen recorder with GIF export
- [GIPHY Capture](https://giphy.com/apps/giphycapture) - Simple GIF recorder
